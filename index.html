About

Seorang Web Developer dengan pengalaman 5 tahun dalam mengembangkan sistem berbasis PHP menggunakan framework Laravel dan Code Igniter, NodeJS mengunakan framework ExpressJS, dan Python pada pengembangan aplikasi optimasi dan modul Odoo ERP.
